# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

The Golf Scanner project takes security seriously. If you discover a security vulnerability, please report it responsibly.

**Do not disclose the vulnerability publicly or to any third parties.**

Please report directly to **antoni@golf.dev** with the following details:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations or fixes

### What to Expect

- **Acknowledgment**: Within 48 hours with an estimated fix timeline
- **Resolution**: We will work to resolve the issue as quickly as possible
- **Disclosure**: Once resolved, we will notify the reporter and acknowledge contributions publicly (anonymity available upon request)
